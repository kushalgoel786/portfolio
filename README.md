### This is my personal portfolio website. 

Built with [Next.js](https://nextjs.org/).

Deployed on Vercel

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
